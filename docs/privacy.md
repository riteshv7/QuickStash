# QuickStash Privacy Policy

Last updated: July 2, 2026

QuickStash is a Chrome extension for capturing ideas, tasks, and notes from your browser.

## Data QuickStash Stores

QuickStash stores the content you create in the extension:

- Captured ideas
- To-do items
- Quick notes
- Source page URL and title for captured ideas
- Theme preference

## How Storage Works

QuickStash stores this data with Chrome extension storage, specifically `chrome.storage.sync`.

If you have Chrome sync enabled, Chrome may sync this extension data through your Google account according to Chrome's own sync behavior and policies. QuickStash does not run its own server and does not send your data to a QuickStash backend.

## Data QuickStash Does Not Collect

QuickStash does not collect:

- Your name or email address
- Account credentials
- Payment information
- Analytics events
- Browsing history
- Advertising identifiers

## Network Requests

QuickStash does not send your notes, tasks, ideas, or captured URLs to an external QuickStash service.

The new tab page loads Google Fonts from `fonts.googleapis.com` and `fonts.gstatic.com` for display styling.

## Permissions

QuickStash asks for Chrome permissions needed to run the extension:

- `storage`: save ideas, tasks, notes, and theme settings.
- `scripting`: inject the capture overlay when you press the shortcut.
- `activeTab`: run the capture overlay on the current tab after a user action.

## Data Deletion

You can delete individual ideas, tasks, and notes inside QuickStash. You can also remove all extension data by uninstalling QuickStash from Chrome or clearing the extension's stored data in Chrome.
