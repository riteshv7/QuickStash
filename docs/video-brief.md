# QuickStash Product Video Brief

Use this with the `product-launch-video` skill.

## Goal

Create a 12-18 second product demo/promo for QuickStash that can be posted on X/Twitter, Reddit, Product Hunt, and the Chrome Web Store listing.

## Format

Primary: portrait `1080x1920`

Secondary export later: square `1080x1080`

## Audience

People who browse all day and lose ideas, links, and tasks before they can sort them.

## Core Message

Capture the thought where it happens. Sort it when you open a new tab.

## Product Claims

- Chrome extension, Manifest V3.
- Shortcut capture from any page.
- New tab Focus Board with Inbox, Today, and Quick Notes.
- No QuickStash account, no app server, no tracking.
- Data uses Chrome extension storage with `chrome.storage.sync`.
- Open source.

## Shot Plan

### Shot 1: Browsing friction

Visual: A browser page with a thought appearing as text near the cursor.

On-screen text: "Idea while browsing?"

Duration: 2 seconds.

### Shot 2: Shortcut capture

Visual: User presses `Cmd+Shift+Y` / `Ctrl+Shift+Y`; QuickStash overlay appears.

On-screen text: "Capture it without switching tabs."

Duration: 4 seconds.

### Shot 3: Save to Inbox

Visual: The typed idea saves, overlay closes, browsing page remains.

On-screen text: "Save the thought. Keep moving."

Duration: 3 seconds.

### Shot 4: New tab triage

Visual: New tab opens to Focus Board. The saved item appears in Inbox beside Today and Quick Notes.

On-screen text: "Sort it on your new tab."

Duration: 5 seconds.

### Shot 5: Trust close

Visual: QuickStash logo/name, GitHub or Web Store link.

On-screen text: "No account. No app server. No tracking."

Duration: 3 seconds.

## Voiceover Option

"When an idea shows up mid-browse, QuickStash lets you catch it without leaving the page. Open a new tab later and sort what matters into Inbox, Today, or Notes. No account, no app server, no tracking."

## Caption Style

Use captions even if voiceover is omitted. Keep each caption under one line on mobile.

## Required Captures

- Capture overlay on a real web page.
- New tab board with realistic sample content.
- Close card with repo URL now, Web Store URL after approval.
